# Student Academic Performance Analysis Using Microsoft Excel

## Project Overview
This project analyzes student academic performance using Microsoft Excel data analysis tools. The aim is to evaluate overall performance, identify grade distribution, detect students who require academic support, and provide recommendations for improving future learning outcomes.

## Objectives
- Calculate overall student performance.
- Analyze score distribution across grades.
- Identify low-performing students requiring support.
- Provide data-driven recommendations for academic improvement.

## Tools Used
- Microsoft Excel
- Excel Formulas
- Pivot Tables
- Charts
- Slicers
- Dashboard

## Analysis Performed
The dataset was cleaned and analyzed by calculating total scores, average scores, grades, remarks, and student rankings. Pivot Tables and charts were created to summarize performance patterns and present insights through an interactive dashboard.

## Key Findings
- Evaluated the overall class performance.
- Identified students performing at different grade levels.
- Highlighted students who need additional academic support.
- Provided strategies for improving future examination results.

## Conclusion
This project demonstrates how Excel can be used to transform student records into meaningful insights that support educational decision-making and improve academic performance.
